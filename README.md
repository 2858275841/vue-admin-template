# vue-admin-template

## 项目介绍

- 使用panjaichen开发的后台管理系统进行二次开发

- 功能涉及到： sku管理 spu管理 权限管理 

- 技术栈：

  ```
  Vue2 axios echarts element-ui Vuex vue-router
  ```
